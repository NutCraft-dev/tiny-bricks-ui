## Props

### size

- Type: `"small" | "medium" | "large"`
- Default: `"medium"`
- Description: Controls the size of the spinner.

### color

- Type: `string`
- Default: `"gray"`
- Description: Customizes the color of the spinner. Use any valid CSS color value.

### speed

- Type: `"slow" | "medium" | "fast"`
- Default: `"medium"`
- Description: Controls the speed of the spinner.
